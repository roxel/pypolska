Python Web Developer

Deployed

25/07/2017 • Kraków

Do obowiązków osoby zatrudnionej na tym stanowisku będzie należało projektowanie, implementacja i testowanie skalowalnych aplikacji internetowych w technologii Python/Django.

Oczekujemy:

  • minimum dwa lata doświadczenia w tworzeniu aplikacji internetowych w Django lub innej technologii webowej
  • bardzo dobrej znajomości Pythona oraz Django
  • dobrej znajomości HTML5, CSS3, ES6
  • znajomości systemu kontroli wersji GIT
  • znajomości linii komend systemu Linux
  • znajomości zagadnień związanych z bazami danych PostgreSQL/Mysql
  • doświadczenia w pisaniu testów jednostkowych
  • zaangażowania i motywacji do pracy nad ciekawymi projektami
  • sumienności, odpowiedzialności, terminowości, umiejętności pisania przejrzystego kodu
  • bardzo dobrej znajomości języka angielskiego
  • umiejętności efektywnego organizowania czasu pracy

Mile widziane:

  • znajomość NoSQL (Elasticsearch, Redis, Mongodb)
  • znajomość AWS
  • znajomość ReactJS
  • znajomość Dockera
  • znajomość Bootstrap
  • znajomość Django Rest Framework, umiejętność tworzenia API
  • znajomość innych obiektowych języków programowania
  • znajomość podstaw administracji Linuxa (w szczególności provisioning/deployment, nginx)

Oferujemy:

  • pracę nad ciekawymi projektami w młodym, dynamicznym i międzynarodowym zespole
  • atrakcyjne wynagrodzenie
  • dobrze wyposażone stanowisko pracy
  • możliwości rozwoju i realizacji własnych pomysłów
  • możliwość pracy zdalnej
  • obiadki firmowe
  • imprezy firmowe
  • budżet konferencyjny
  • artystyczny avatar For The Win :)

Jeśli uważasz, że spełniasz powyższe warunki wyślij CV, listę swoich projektów lub adres swojego internetowego portfolio (linki, github, bitbucket) wraz z klauzulą:
