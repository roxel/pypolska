Dołącz

Cele stowarzyszenia

  • Pomoc w organizacji wydarzeń poświęconych językowi programowania Python i powiązanym technologiom,

  • Działalność promocyjna i edukacyjna w zakresie języka Python i powiązanych technologii,

  • Integracja lokalnych grup społecznościowych, których głównym tematem spotkań jest język programowania Python lub powiązane z nim technologie,

  • Popularyzacja języka programowania Python wśród programistów,

  • Wsparcie w rozwoju rynku pracy związanej z językiem Python,

  • Pomoc w rozwoju zawodowym programistów Pythona,

  • Wsparcie projektów związanych z Pythonem i powiązanymi technologiami,

  • Współpraca z innymi organizacjami pozarządowymi i podmiotami zainteresowanymi językiem Python, w tym z Python Software Foundation.

Dołącz do stowarzyszenia

Dołączenie do stowarzyszenia jest wyjątkową okazją do działalności w polskiej społeczności programistów Pythona. Dodatkową korzyścią są niższe koszty udziału w wydarzeniach np. PyConie.

Dołączenie jest możliwe poprzez wypełnienie deklaracji członkowskiej.
